Baixe esta versão do Python
https://www.python.org/downloads/release/python-2715/

Depois de instalar, rode os seguintes comandos no terminal:
pip install pyserial
pip install matplotlib

Abra o editor "IDLE", que está instalado na pasta da Python
Com o IDLE, abra o arquivo "SerialPlot.py" e altere a porta do arduino
Rode o programa (Run > Run Module F5)
